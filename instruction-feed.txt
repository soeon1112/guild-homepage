메인 페이지에 최신 현황 피드를 추가해줘. 위치: 흔적 남기기 아래, 메뉴 버튼 그리드 위.

1) Firestore 컬렉션: "activity"
필드: type(string), nickname(string), message(string), createdAt(timestamp)

2) 활동 기록 자동 생성 - 아래 상황이 발생할 때마다 activity 컬렉션에 문서 추가:
- 미니홈피 프로필 사진 수정 시: "{nickname}님이 프로필 사진을 수정했습니다"
- 미니홈피 한마디 수정 시: "{nickname}님이 한마디를 수정했습니다"
- 미니홈피 사진첩에 사진 올릴 때: "{nickname}님의 공간이 업데이트되었습니다"
- 미니홈피 방명록 남길 때: "{nickname}님이 방명록을 남겼습니다"
- 앨범에 새 사진 업로드 시: "새 앨범 사진이 업로드되었습니다"
- 정보 게시판에 새 글 등록 시: "정보 게시판에 새 글이 등록되었습니다: {제목}"

3) 메인 페이지 표시:
- "최신 현황" 제목 (파란색 #4a90d9, 흰색 외곽선)
- 최근 10개를 시간 역순으로 표시
- 각 항목: 메시지 + 시간 (오늘이면 HH:MM, 아니면 MM/DD HH:MM)
- 실시간 업데이트: onSnapshot 사용

4) 디자인:
- 배경 rgba(0,0,0,0.4) 반투명, 둥근 모서리
- 메시지 텍스트 흰색, 닉네임 부분만 파란색
- 시간 텍스트 연한 회색
- 모바일 퍼스트

이미지에 filter 적용하지 마.
