2) 한마디 컴포넌트:
위치: 로고 아래, 메뉴 버튼 그리드 위에 배치.
- "오늘의 한마디" 제목
- 입력 영역: 닉네임 input + 한마디 input + 등록 버튼을 가로로 배치. 모바일에서는 세로로.
- 한마디 목록: 최근 5개만 시간 역순으로 표시. 각 항목은 "닉네임: 한마디 (시간)" 형태.
- Firestore 컬렉션 이름: "guestbook"
- 각 문서 필드: nickname(string), message(string), createdAt(timestamp)
- Firestore 쿼리에서 limit 5로.

3) 디자인:
- 입력 영역 배경: rgba(0,0,0,0.4) 반투명, 둥근 모서리
- 입력 필드: 배경 rgba(255,255,255,0.1), 흰색 텍스트, placeholder도 연한 흰색
- 등록 버튼: 금색 테두리, 금색 텍스트
- 한마디 목록: 각 항목 사이 연한 구분선, 흰색 텍스트, 닉네임은 금색
- 시간 표시: 오늘이면 "HH:MM", 오늘이 아니면 "MM/DD HH:MM"
- 전체적으로 메인 페이지 분위기에 맞게

4) 기능:
- 로그인 불필요. 누구나 바로 작성 가능.
- 빈 닉네임이나 빈 한마디면 등록 안 되게
- 등록 성공하면 입력 필드 초기화
- 실시간 업데이트: onSnapshot으로 새 한마디가 바로 보이게
- "use client" 컴포넌트로 만들어야 해

이미지에 filter 적용하지 마.
