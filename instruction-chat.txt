홈페이지 오른쪽 하단에 길드 채팅 기능을 추가해줘.

1) 채팅 버튼:
- 오른쪽 하단에 동그란 채팅 아이콘 버튼 고정 (position fixed).
- 클릭하면 채팅창 열리고 닫히고 토글.
- 로그인 안 한 상태면 "로그인이 필요합니다" 표시.
- 버튼 스타일: 배경 rgba(0,0,0,0.7), 크기 56px, 채팅 아이콘은 lucide-react MessageCircle 사용, 금색.

2) 채팅창:
- 오른쪽 하단에 고정. 모바일: 화면 전체, PC: 가로 360px 세로 500px.
- 상단: "길드 채팅" 제목 + 닫기 X 버튼.
- 중간: 메시지 목록 영역. 스크롤 가능. 최신 메시지가 아래에.
- 하단: 메시지 입력칸 + 전송 버튼.

3) Firestore:
- 컬렉션: "chat"
- 필드: nickname(string), message(string), createdAt(timestamp)
- onSnapshot으로 실시간 업데이트. 새 메시지 오면 자동 스크롤.
- 최근 50개 메시지만 불러오기.

4) 메시지 표시:
- 내 메시지: 우측 정렬, 파란색 배경 rgba(74,144,217,0.3)
- 다른 사람 메시지: 좌측 정렬, 배경 rgba(255,255,255,0.1)
- 닉네임 파란색 #4a90d9 작게, 메시지 흰색, 시간 연한 회색 작게.
- 닉네임 클릭 시 해당 미니홈피로 이동 가능 (기존 닉네임 클릭 기능과 동일).

5) 채팅창 디자인:
- 배경 rgba(0,0,0,0.85)
- 둥근 모서리
- 입력칸: 배경 rgba(255,255,255,0.1), 흰색 텍스트
- 전송 버튼: 금색

6) 채팅 포인트: 메시지 보내기에는 포인트 없음.

7) 모든 페이지에서 보이게. 채팅 버튼은 항상 고정.

이미지에 filter 적용하지 마.
