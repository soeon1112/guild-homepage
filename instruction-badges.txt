미니홈피에 배지 컬렉션 시스템을 추가해줘.

1) 배지 컬렉션 위치: 미니홈피 프로필 영역 아래. 기본 접혀있고 "배지 컬렉션 (획득수/전체수)" 표시. 펼치기 누르면 전체 배지 보이게.

2) 배지 표시: 동그란 원 그리드 (모바일 5열, 데스크탑 8열). 미획득: 회색 원에 ❓. 획득: 유채색 배경에 이모지. 배지 터치 시 팝업: 배지 이름, 조건 설명, 획득 날짜(획득한 경우). 히든 배지는 미획득 시 조건도 "???"로 표시.

3) Firestore: users/{nickname}/badges 서브컬렉션. 각 문서: badgeId(string), earnedAt(timestamp).

4) 배지 체크: 활동할 때마다 조건 확인. 달성 시 배지 부여 + 화면에 "새로운 배지를 획득했습니다: {배지이름}" 토스트 알림 3초 표시.

5) 중요 - 기존 활동 소급 적용: 배지 시스템 적용 시 기존 데이터를 기반으로 이미 조건을 달성한 사용자에게 자동으로 배지를 부여해줘. 구체적으로:
- 로그인 시 해당 유저의 기존 활동 데이터를 Firestore에서 확인해서 아직 안 받은 배지 중 조건 달성한 것 자동 부여.
- 예: 이미 댓글 15개 쓴 사람이 로그인하면 first_login, comment_10 배지 자동 획득.
- 예: 이미 프로필 등록한 사람은 first_profile 배지 자동 획득.
- 예: 이미 방명록 12개 남긴 사람은 guestbook_10 배지 자동 획득.
- Firestore의 기존 컬렉션들(guestbook, board, album, chat, members 등)을 조회해서 각 유저의 활동 수를 카운트하고 조건 충족 시 배지 부여.
- 이 소급 체크는 로그인 시 한 번만 실행하고, 이후에는 실시간 활동 기반으로 체크.
